# Screenshot Placeholders

Add the following screenshots to this directory:

1. `main_menu.png` - Screenshot of the main launcher menu with 4 options
2. `dmx_address.png` - Screenshot of the DMX address programming interface
3. `static_look.png` - Screenshot of the static look configuration with visual channel bars
4. `flash_boot.png` - Screenshot of the flash boot toggle interface

To take screenshots:
1. Run the firmware on your actual hardware
2. Capture the screen using a camera or screenshot tool
3. Crop and resize the images to appropriate dimensions (recommend 320x240)
4. Save as PNG files using the filenames above

Once screenshots are added, you can remove this file.
